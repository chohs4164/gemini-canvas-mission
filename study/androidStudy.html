Patterns Comparison

핵심 아키텍처 비교

상황에 맞는 최적의 디자인 패턴을 선택하세요.

Pattern

MVC

Activity가 모든 것을 제어하는 초기 방식입니다. 코드가 비대해지기 쉬워 현대적인 앱에는 권장되지 않습니다.

Best Practice

MVVM

ViewModel이 UI 상태를 보유하고 View는 이를 구독합니다. 테스트가 용이하고 생명주기에 안전합니다.

Modern

MVI

단방향 데이터 흐름(UDF)을 보장합니다. 상태 관리가 명확하며 Jetpack Compose와 함께 사용할 때 시너지가 큽니다.

Design System

Material Design 3

Dynamic Color

사용자의 배경화면에서 색상을 추출하여 개인화된 테마를 생성합니다.

Adaptive Layout

다양한 화면 크기(창 크기 클래스)에 유연하게 대응합니다.
